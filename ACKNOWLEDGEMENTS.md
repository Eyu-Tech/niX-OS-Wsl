# Acknowledgements

This configuration stands on the shoulders of people who built things worth
standing on. Not a legal notice — just genuine credit.

## The infrastructure

- **[NixOS-WSL](https://github.com/nix-community/NixOS-WSL)** — the module that
  makes any of this possible under Windows. Remarkable piece of work.
- **[nixpkgs](https://github.com/NixOS/nixpkgs)** — the package collection that
  makes Nix actually useful. Maintained by thousands of contributors.
- **[home-manager](https://github.com/nix-community/home-manager)** — declarative
  user environments done right.
- **[fenix](https://github.com/nix-community/fenix)** — Rust toolchains in Nix,
  without the pain.

## The tools I actually love using

- **[Rust](https://www.rust-lang.org)** — a language that treats you like an adult.
  The compiler catches what you miss, the type system keeps the model honest, and
  pattern matching is just home. Built for minds that like structure.
- **[Neovim](https://neovim.io)** — editor as instrument, not just tool.
- **[Starship](https://starship.rs)** — prompt that stays out of the way until
  it matters.
- **[zellij](https://zellij.dev)** — terminal multiplexer that finally got the UX right.
- **[yazi](https://github.com/sxyazi/yazi)** — file manager fast enough to not
  break your flow.
